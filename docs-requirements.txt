# Caminho relativo: docs-requirements.txt
#
# Ferramentas Python para gerar o site de documentação (mkdocs.yml, docs/) —
# não faz parte do binário Rust, só do fluxo de documentação.
#
# Uso:
#   uv venv .venv-docs && uv pip install --python .venv-docs/bin/python -r docs-requirements.txt
#   .venv-docs/bin/mkdocs serve      # servidor local com live-reload
#   .venv-docs/bin/mkdocs build      # gera ./site (estático, ignorado pelo git)
#
# (ou, sem uv: pip install -r docs-requirements.txt dentro de qualquer venv Python 3.10+;
# em Debian/Ubuntu, `sudo apt install mkdocs-material` também resolve as duas dependências.)
